High-performance Python web framework providing RESTful API endpoints for chat and document upload operations. Handles CORS, request validation, and error handling.

LangGraph Agent System
Multi-agent orchestration using LangGraph. Implements a stateful graph with conditional routing between specialized agents.
LangGraph LangChain
Supervisor Agent
Central coordinator that analyzes user requests and routes tasks to appropriate specialized agents (Researcher or Writer).
Researcher Agent
Specialized agent that performs RAG operations, querying the vector store to retrieve relevant context from uploaded documents.
Writer Agent
Generates comprehensive responses using the LLM, incorporating context provided by the Researcher agent.
Vector Store Service
Manages document ingestion, text splitting, embedding generation, and semantic search using ChromaDB.
ChromaDB HuggingFace

📊 Data Flow Process
Document Upload
User uploads PDF documents through the frontend interface. Files are sent to the backend via POST request.
Text Extraction
PyPDFLoader extracts text content from the uploaded PDF documents, preparing it for processing.
Text Splitting
RecursiveCharacterTextSplitter breaks documents into chunks (1000 characters with 200 character overlap).
Embedding Generation
HuggingFace all-MiniLM-L6-v2 model generates semantic embeddings for each text chunk.
Vector Storage
Embeddings are stored in ChromaDB for persistent storage and fast similarity search.
Query Processing
When user asks a question, the Researcher agent queries ChromaDB to find relevant document chunks.
Context Retrieval
Most relevant document chunks are retrieved and provided as context to the Writer agent.
Response Generation
Writer agent uses Llama 3 via Ollama to generate a comprehensive response based on the retrieved context.
